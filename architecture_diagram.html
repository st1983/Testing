📊 Test Healing Execution Flow

  1. Test Execution - TestNG framework runs test case, WebDriver attempts to locate element
  2. Failure Detection - NoSuchElementException or timeout triggers healing mechanism
  3. DOM Snapshot - Current DOM state is captured and compared with historical snapshots
  4. Change Analysis - LocatorAnalyzer identifies what changed in the DOM structure
  5. Feature Extraction - ML features computed: path similarity, attribute matching, text content, etc.
  6. ML Prediction - Random Forest model predicts optimal healing strategy
  7. Strategy Execution - Recommended strategy applied: path recovery, attribute matching, fuzzy matching, or semantic matching
  8. Validation - If element found and matches characteristics, healing succeeds
  9. Fallback Handling - If primary strategy fails, fallback strategies are attempted
  10. Learning Recording - Success/failure recorded for continuous model improvement
  11. Test Continuation - Test resumes with healed element, or fails with detailed diagnostics

🔧 Five Healing Strategies

1. Path Recovery

Generates XPath variations by removing indices, using contains clauses, and exploring alternative paths. Effective for simple DOM restructuring.

Success Rate: 55%

2. Attribute Matching

Locates elements using ID, class names, data attributes, and text content. Works well when element structure remains stable.

Success Rate: 65%

3. Fuzzy Matching

Uses string similarity algorithms (Levenshtein distance) on class names and text content. Handles minor UI text changes.

Success Rate: 70%

4. Semantic Matching

Leverages ML-generated embeddings to find elements with highest cosine similarity. Best for complex structural changes.

Success Rate: 82%

5. Computer Vision

Visual similarity matching using image feature detection (SIFT/ORB). Ideal for screenshot-based comparison.

Success Rate: 78%

Fallback Chain

When primary strategy fails, ML model determines next best strategy. Maximizes healing success through intelligent fallback ordering.

Combined Success: 95%
